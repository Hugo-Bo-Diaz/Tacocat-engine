What is TacocaT engine?

Tacocat engine is a 3D engine made for the subject of 3D engines of the polytechnic university of catalonia imparted by Ricard Pillosu

Our Team

  • Hugo Bo

    Lead programmer
    Has done most of thw workload, as he has focused on the core parts of the engine

    Daniel Garcia

    Lead artist
    Has done quadtree, hierarchy, UI in engine, website

Main subsystems:

Importer
Can drag and drop multiple file formats into the engine in order to load them
Camera
Camera movement using WASD keys and right mouse or Alt + left mouse to orbit camera
Meshes and textures
Can load and display 3d objects and textures
Hierarchy
Display all loaded objects in scene with a hierarchy
Transform
Can modify position rotation and scale of gameobjects
Mouse pick
Can select gameobjects using mouse picks
Quadtree
Static gameobjects are selected more quickly using a quadtree
Camera FOV
Gameobjects outside the field of view are not loaded
Window resize
Resize window using borders
Bounding boxes
All objects are surrounded by bounding boxes in order to ease selection and visualization
Scene serialization
Save scene to be able to load it later
Own fileformat
All assets used are converted into our own fileformat to speed up the loading process
Time management
Can play pause stop scene in order to make adjustments

Wwise

For the 3rd delivery we had to implement Wwise, a middleware that eases the process of integrating audio into a 3d engine
